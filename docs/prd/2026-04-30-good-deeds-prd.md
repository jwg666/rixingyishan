# PRD（总览索引）｜日行一善（Good Deeds）（MVP：账号+云同步）

本仓库拆分为两个子工程：
- **前端**：`rixingyishan-ui/`（uni-app + Vue3 + TS）
- **后端**：`rixingyishan-service/`（登录/记录/上传凭证/日历打点）

## PRD 拆分
- **PRD（前端）**：`docs/prd/2026-04-30-good-deeds-prd-ui.md`
- **PRD（后端）**：`docs/prd/2026-04-30-good-deeds-prd-service.md`
- **实现差距（前后端拆分）**：`docs/prd/2026-04-30-implementation-gap.md`

## MVP 总目标（摘要）
- 记录：文字/照片/视频，按天回顾（日历打点 + 当天列表 + 详情）
- 账号：手机号验证码登录（全端一致）
- 同步：媒体上传 + 元数据同步 + 弱网重试/恢复
- 退出策略：**固定方案 A**（退出仅清账号态，本地记录保留；再次登录提示合并上传）

## MVP 验收用例（摘要）
- 未登录可本地创建记录；登录后提示合并上传
- 新建后返回首页：列表与日历打点刷新
- 媒体上传失败可重试；重启/回前台可恢复队列
- 小程序真机上传可用（配置合法域名后）
